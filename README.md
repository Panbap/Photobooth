# 📸 Photobooth Online

> 🎞️ Ứng dụng chụp ảnh trực tiếp trên trình duyệt – nhanh, tiện và không cần cài đặt.

🌐 **Live Demo:** https://panbap.github.io/Photobooth/

---

## 🚀 Giới thiệu

Đây là một project **Photobooth Online** được xây dựng bằng HTML, CSS và JavaScript.

Ứng dụng cho phép bạn sử dụng **camera (webcam)** để chụp ảnh trực tiếp ngay trên trình duyệt.

💡 Không cần phần mềm – chỉ cần mở web là chụp!

---

## 🎯 Cách hoạt động

- Cho phép truy cập camera  
- Hiển thị video trực tiếp từ webcam  
- Nhấn nút để chụp ảnh  
- Hiển thị ảnh vừa chụp  

---

## ✨ Tính năng

- 📷 Chụp ảnh trực tiếp từ webcam  
- ⚡ Hiển thị ảnh ngay lập tức  
- 🎨 Giao diện đơn giản, dễ dùng  
- 🌐 Chạy hoàn toàn trên browser  
- 🔒 Sử dụng camera qua Web API  

---

## 🛠️ Công nghệ sử dụng

- HTML5 (MediaDevices API)  
- CSS3  
- JavaScript

## ⚠️ Lưu ý

- Trình duyệt sẽ yêu cầu quyền truy cập camera
